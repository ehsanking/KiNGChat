# کیانگ‌چت 👑
### پیام‌رسان امن برای عصر خصوصی

[نسخهٔ انگلیسی](README.md)

کیانگ‌چت یک پیام‌رسان متن‌باز و امن است که برای محافظت از حریم خصوصی شما طراحی شده‌است. این برنامه از نکس‌جی‌اس و ریاکت در فرانت‌اند و نود.جی‌اس در بک‌اند بهره می‌برد، داده‌ها را با پایگاه‌داده PostgreSQL و ORM پریزما ذخیره می‌کند و با Socket.IO ارتباط لحظه‌ای برقرار می‌کند. همهٔ پیام‌ها در مرورگر کاربر با استفاده از Web Crypto API (تبادل کلید ECDH P‑256، مشتق‌سازی HKDF‑SHA256، و رمزنگاری AES‑256‑GCM) رمزنگاری می‌شوند و فقط گیرنده‌ی اصلی قادر به رمزگشایی آن‌ها است.

## ویژگی‌ها

- **رمزنگاری سرتاسری (E2EE):** کلیدها از طریق ECDH تبادل و با HKDF مشتق می‌شوند؛ پیام‌ها با AES‑256‑GCM رمزنگاری می‌شوند.
- **گفتگوهای بلادرنگ:** ارسال و دریافت فوری پیام‌ها، گروه‌ها و کانال‌ها با استفاده از Socket.IO و قابلیت مقیاس‌پذیری با Redis.
- **سیستم مخاطبین:** جستجو و افزودن کاربران براساس نام کاربری یا شناسه‌ی عددی و ذخیره‌ی فهرست مخاطبین.
- **گروه‌ها و کانال‌ها:** ایجاد گروه‌های خصوصی و عمومی با لینک دعوت و نقش‌های مالک، مدیر، ناظر و عضو.
- **احراز هویت دو مرحله‌ای:** فعال‌سازی، تأیید و غیرفعال‌سازی ۲FA مبتنی بر TOTP (Google Authenticator یا Authy).
- **پایگاه داده و تاریخچه:** نگهداری پیام‌ها و فایل‌ها در پایگاه‌داده PostgreSQL و بارگذاری تاریخچه با صفحه‌بندی.
- **پنل مدیریت:** مدیریت کاربران، تنظیمات سامانه، ثبت فعالیت‌ها، محدودیت‌ها و عملیات مدیریتی.
- **اپلیکیشن پیش‌رونده (PWA):** قابلیت نصب به‌عنوان اپلیکیشن موبایل یا دسکتاپ و کارکرد آفلاین.
- **اسکریپت نصب تک‌خطی:** بررسی پیش‌نیازها، تولید کلیدها، راه‌اندازی Docker و Caddy و اجرای مهاجرت‌های پایگاه‌داده.
- **استقرار داکر:** فایل Dockerfile و docker‑compose رسمی برای استقرار تولیدی با SSL خودکار.
- **رابط کاربری مدرن:** طراحی زیبا با Tailwind CSS، آیکن‌های Lucide و انیمیشن‌های روان.

## نصب سریع
```bash
curl -fsSL https://raw.githubusercontent.com/ehsanking/KiNGChat/main/install.sh | bash
```

## نصب دستی
```bash
git clone https://github.com/ehsanking/KiNGChat.git
cd KiNGChat
cp .env.example .env
npm install --legacy-peer-deps
npm run build
npm test  # اختیاری
```
فایل `.env` را ویرایش کنید و سپس با یکی از روش‌های زیر اجرا کنید:
```bash
npm run dev            # اجرا در حالت توسعه
npm start             # اجرا در حالت تولید
docker compose up -d --build   # اجرا با Docker
```

### استفاده از آینه‌ها
اگر به دلیل تحریم یا محدودیت، دسترسی به مخزن اصلی NPM ممکن نیست، می‌توانید یک آینه تعریف کنید:
```bash
npm config set registry https://registry.npmmirror.com
# یا برای Yarn:
yarn config set registry https://registry.npmmirror.com
```
