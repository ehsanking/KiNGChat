# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **Do not** open a public issue.
2. Email the maintainer or open a private security advisory via GitHub's Security tab.
3. Include a description of the vulnerability, steps to reproduce, and potential impact.
4. You can expect an initial response within 72 hours.
5. We will coordinate a fix and disclosure timeline with you.

Thank you for helping keep Elahe Messenger safe.
